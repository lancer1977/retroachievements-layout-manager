## Boundaries

This repository is a legacy Windows desktop tool for RetroAchievements streaming
overlays. Treat the checked-in source as a .NET Framework 4.7.2 WinForms and
ClickOnce project; do not assume the stewardship Linux environment can compile
or run the full app.
